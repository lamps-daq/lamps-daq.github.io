Linux Advanced MultiParameter System


LAMPS is a data acquisition and analysis package that supports VME, CAMAC-FERA and a number of CAMAC controllers. It can also be used for offline data analysis. It runs under Linux (any recent version) and in cygwin under MS Windows. It is designed for large-scale experiments.

It is in use at the BARC-TIFR Pelletron Laboratory, since August 2002.
